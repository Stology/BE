# 구현·검토 가이드

가이드는 현재 코드에서 관찰한 관례와 팀이 합의해 적용할 검토 기준을 구분합니다. 작업 정책의 정본은 [AGENTS.md](../../AGENTS.md)이고, 문서 전체 경로는 [문서 인덱스](../README.md)에서 확인합니다.

- [코드 관례](code-conventions.md): `src/main/java/com/stology/be`에서 관찰한 배치와 표현
- [PR 검토](pr-review.md): 자동 확인과 사람이 판단할 항목

도메인별 현재 모델과 탐색 경로는 [도메인 온보딩](../onboarding/domain/README.md)을 기준으로 합니다.
